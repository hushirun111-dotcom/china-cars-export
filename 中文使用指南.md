# Careal 中文快速使用指南

## 🚀 立即开始

### 第一步：安装

```bash
npm install
```

### 第二步：配置

创建 `.env.local` 文件，添加以下内容：

```env
# 必填项
NEXT_PUBLIC_WHATSAPP_NUMBER=你的WhatsApp号码
NEXT_PUBLIC_CONTACT_EMAIL=你的邮箱
ADMIN_PASSWORD=管理员密码（请修改！）

# 可选项
NEXT_PUBLIC_GA_ID=你的Google Analytics ID
```

### 第三步：启动

```bash
npm run dev
```

打开浏览器访问：http://localhost:3000

## 📱 功能说明

### 1. 询价系统

**客户端：**
- 访问任意车辆详情页
- 填写询价表单
- 提交后自动保存

**管理端：**
- 访问：`/en/admin/inquiries`
- 输入密码登录
- 查看所有客户询价

### 2. 在线聊天

**WhatsApp按钮（默认）：**
- 网站右下角绿色按钮
- 点击直接打开WhatsApp
- 修改号码：`.env.local` → `NEXT_PUBLIC_WHATSAPP_NUMBER`

**Tawk.to客服（可选）：**
- 注册：https://www.tawk.to
- 配置Property ID到 `.env.local`
- 详见 `FEATURES-CONFIG.md`

### 3. 管理面板

**访问地址：**
```
http://localhost:3000/en/admin/inquiries
```

**默认密码：** admin123（请立即修改！）

**功能：**
- 查看所有询价
- 管理询价状态
- WhatsApp/邮件快速联系
- 复制客户信息

## 📋 常见问题

### Q: 询价提交失败？
A: 检查 `data/inquiries/` 目录是否存在
```bash
mkdir -p data/inquiries
```

### Q: 管理面板登录不了？
A: 确认 `.env.local` 中设置了 `ADMIN_PASSWORD`

### Q: WhatsApp按钮不显示？
A: 检查 `NEXT_PUBLIC_WHATSAPP_NUMBER` 是否设置

### Q: 如何备份数据？
A: 定期复制 `data/inquiries/` 目录
```bash
cp -r data/inquiries data/inquiries_backup
```
